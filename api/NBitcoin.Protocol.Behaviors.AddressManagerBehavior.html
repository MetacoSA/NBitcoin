Class AddressManagerBehavior

The AddressManagerBehavior class will respond to getaddr and register advertised nodes from addr messages to the AddressManager. The AddressManagerBehavior will also receive feedback about connection attempt and success of discovered peers to the AddressManager, so it can be used later to find valid peer faster.
